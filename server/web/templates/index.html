ICP备案查询系统 (Go)

支持域名、App、小程序、快应用的 ICP 备案与违规查询，批量并发，历史导出。

CLI 命令
  • icpcli query baidu.com — 查询单个域名
  • icpcli query 微信 -t app — 查询 App 备案
  • icpcli batch -f domains.txt -t web — 批量查询
  • icpcli serve — 启动 Web 服务
  • icpcli mcp — 启动 MCP Server
